14 Online Forms

ASM allows you to setup online forms that you can use to take information from members of the public through your website (if your ASM is publically accessible, or you are using sheltermanager.com).

This is very useful for things like adoption and waiting list application forms, questionnaires, taking lost or found animal information from people, etc. They can be used for any situation where you'd like to take information from web site visitors.

Forms need a name and a description. The description will be shown to the user while they are filling out the form. You can also specify a page you'd like to redirect the user to after they've completed the form.

If you use certain key fields (which the system will autocomplete for you), the system can create person, lost animal, found animal or waiting list records directly from submitted form data. If you set some person flags on your form, any person record created from the form data will automatically have those flags.

As you can see in the screenshot, each form has a “Form URL”. This is the web link you can use on your website to link people to the form. You can also click it directly there in the UI to test your form. Note that sheltermanager.com uses a short term 2 minute cache on forms, so if you make changes to a form you've recently viewed, you may have to wait 2 minutes for any changes you make to appear.

Clicking on the form's name will allow you to edit the individual fields of information the report will take.

The dialog will autocomplete the known fields that ASM can look for when importing records (documented in an appendix at the back of this manual).

To create any record, you will need at least a lastname or surname field. Lost animal records need an arealost and description, found animal records need an areafound and description, waiting list records need a description.

When forms are submitted through the website they come through to the “View Incoming Forms” screen, where the values can be inspected by clicking the name of the form submission. The screen shows a preview of the incoming data and the IP address that submitted it.

Selecting a form allows you to attach it to an existing person reecord (with the attach button), create a new person record from the data, or create a new lost or found animal and waiting list entry. In the cases of lost and found animals and waiting list entries they need a contact, so there must be enough information in the form to create the person record to go with them. You do not need to create a person record and then the other type of record as creating a lost or found animal or waiting list record will automatically create the person as well.

When you create a new record or attach the form, the whole form will be included in the media tab of any created records (person AND lost/found animal or waiting list). The screen will put a link in the Link column to give you a clickable link to the newly created record as well so you can view it.

You can safely delete incoming forms once they have been attached to a record. The system will also remove incoming forms older than 2 weeks by default.
